Detailed Description

Computes the osculating Keplerian elements from the satellite state vector for elliptic orbits.

This function computes the osculating Keplerian elements (semilatus rectum, semimajor axis, eccentricity, inclination, longitude of the ascending node, argument of pericenter, and mean anomaly) from the satellite state vector for elliptic orbits.

Parameters
yState vector (x, y, z, vx, vy, vz) in meters and meters per second.
pSemilatus rectum in meters.
aSemimajor axis in meters.
eEccentricity.
iInclination in radians.
OmegaLongitude of the ascending node in radians.
omegaArgument of pericenter in radians.
MMean anomaly in radians.
Note
The function cannot be used with state vectors describing a circular or non-inclined orbit.

void
