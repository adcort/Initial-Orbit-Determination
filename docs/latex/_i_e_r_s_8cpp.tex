\doxysubsection{Detailed Description}
IERS\+: Management of IERS time and polar motion data. 

This function retrieves IERS Earth rotation parameters based on the input Modified Julian Date (Mjd\+\_\+\+UTC) and the provided Earth Orientation Parameters (eop).

\begin{DoxyNote}{Note}
The function assumes the input eop matrix is structured such that each row contains the following parameters in sequence\+: MJD, x\+\_\+pole, y\+\_\+pole, UT1-\/\+UTC, LOD, dpsi, deps, dx\+\_\+pole, dy\+\_\+pole, TAI-\/\+UTC.

If interp is \textquotesingle{}l\textquotesingle{}, linear interpolation is used between nearest dates in eop. If interp is \textquotesingle{}n\textquotesingle{} or any other character, the nearest date\textquotesingle{}s data from eop is used.
\end{DoxyNote}
