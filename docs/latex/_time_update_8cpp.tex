\doxysubsection{Detailed Description}
Performs a time update on the state covariance matrix. 

This function performs a time update on the state covariance matrix {\ttfamily P} using the state transition matrix {\ttfamily Phi} and a constant {\ttfamily Qdt} representing the process noise. If only {\ttfamily P} and {\ttfamily Phi} are provided, {\ttfamily Qdt} defaults to {\ttfamily 0.\+0}.
